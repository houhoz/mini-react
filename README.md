# 实现 mini react

实现目标如下：

1. createElement；
2. render；
3. 并发模式；
4. Fibers；
5. 渲染和提交 ；
6. 协调；
7. 函数组件；
8. hooks；
9. 类组件

安装依赖

```bash
npm install --save-dev babel-loader @babel/core
npm install webpack webpack-cli --save-dev
npm install --save-dev @babel/preset-react
npm install --save-dev html-webpack-plugin
```
